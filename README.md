# Duplicate Cleaner

Aplikasi desktop berbasis Python (Tkinter) yang berfungsi untuk membersihkan data duplikat dengan mudah dan cepat. Aplikasi ini memiliki dua fitur utama: menghapus duplikat internal dari satu daftar data, atau menghapus item berdasarkan daftar pembanding (Kolom 2).

## Features
* **Hapus Duplikat**: Mengeliminasi baris yang sama dari data utama secara otomatis dan mengurutkannya (`A-Z`).
* **Hapus via Pembanding**: Menghapus item di Data Utama jika item tersebut eksis di Data Pembanding.
* **Real-time Counter**: Menghitung jumlah baris/item secara otomatis saat Anda mengetik atau menempelkan data.
* **User Friendly UI**: Antarmuka bersih dengan visual indikator warna untuk hasil bersih (hijau) dan data terhapus (merah).

## Preview Layout
Aplikasi ini berjalan di mode desktop dengan ukuran layar tetap 400x700 piksel, sangat cocok untuk disandingkan di sebelah layar kerja Anda.

## Prerequisites
Aplikasi ini menggunakan modul bawaan Python (**Tkinter**). Anda hanya perlu menginstal Python versi 3.x di komputer Anda.

## How to Run
1. Clone repositori ini:
   ```bash
   git clone [https://github.com/qwertyaqu-prog/Duplicate-Cleaner.git](https://github.com/qwertyaqu-prog/Duplicate-Cleaner.git)
   ```
2. Masuk ke direktori proyek:
   ```Bash
   cd Duplicate-Cleaner
   ```
3. Jalankan aplikasi:
   ```Bash
   python Duplicate-Cleaner.py
   ```
